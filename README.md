# 📚 Library Automation System (C)

A console-based **Library Automation System** developed in **C**. This project demonstrates the use of fundamental data structures, dynamic memory management, file operations, and modular programming by simulating a real-world library management system.

---

## 🚀 Features

### 👨‍🎓 Student Management
- Add student
- Delete student
- Update student information
- List all students
- Search student details
- View borrowing history
- List students with penalties
- List students who have not returned books

### 📖 Book Management
- Add new books
- Delete books
- Update book information
- Search books
- List available books
- Borrow books
- Return books
- Automatic copy (label) generation

### ✍️ Author Management
- Add author
- Delete author
- Update author information
- View author details
- List all authors
- Match books with authors

### 📂 File Management
- CSV file storage
- Binary file storage
- Automatic data loading
- Automatic data saving
- Persistent records

### ⏰ Borrowing System
- Borrow books
- Return books
- Late return detection
- Automatic penalty point deduction
- Borrowing history tracking

## 💾 Data Storage

The system stores its data using:

- **CSV Files**
  - Students
  - Authors
  - Books
  - Borrowing Records

- **Binary File**
  - Book-Author relationships

Data is automatically loaded when the program starts and saved after every update.
